== Description ==

Creates new shipping method based on 'sendbox API' and posts shipment under order actions in Admin.

== Installation ==

1. Upload `woocommerce-shipping-sendbox` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
